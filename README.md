# Natasha Caraos Portfolio

Welcome to my personal portfolio website, showcasing my work as a designer. The site highlights human-centered design projects, demonstrates my skills, and allows visitors to book a free consultation.

## Live Site

Check out the live portfolio here: [Live Portfolio](https://natorbit.github.io/everyman-portfolio/)

## Features

- **Hero Section** with an animated profile photo and introduction.
- **Persuasion Section** implementing Cialdini's principles:
  - **Reciprocity:** Free audit/consultation.
  - **Scarcity:** Limited availability messaging.
  - **Authority:** Testimonials and case studies.
  - **Social Proof:** Trusted by startups and brands.
- **Projects Section** with interactive hover overlays on images.
- **Contact Form** integrated with Zapier for easy submissions.
- Fully **responsive design**, works on mobile, tablet, and desktop.

## Directory Structure

- `index.html` – Main HTML file containing all sections.
- `images/` – Folder containing your hero image and project screenshots.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (Calendly integration, form handling)
- Zapier Webhooks (for contact form submissions)
